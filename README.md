# Ebook 캡처 도구

Windows에서 지정한 화면 영역을 반복적으로 스크린샷 찍어 파일로 저장합니다.

---

## 설치

Python 3.8 이상이 필요합니다.

```bash
pip install -r requirements.txt
```

---

## 실행

```bash
python capture.py
```

---

## 사용 방법

### 1단계 — 캡처 영역 선택

**영역 선택** 버튼을 클릭하면 화면이 어두워지면서 오버레이가 나타납니다.  
마우스로 캡처할 영역(ebook 페이지)을 드래그하세요.  
`ESC`를 누르면 취소합니다.

### 2단계 — 저장 설정

| 항목 | 설명 |
|------|------|
| 저장 폴더 | 이미지를 저장할 폴더 경로 (기본: `~/Pictures/ebook_capture`) |
| 파일명 접두사 | 파일 이름 앞에 붙는 텍스트 (기본: `page`) |
| 시작 번호 | 첫 번째 파일의 번호 (기본: `1`) |
| 형식 | `png` (무손실) 또는 `jpg` (용량 작음) |

저장 파일명 예: `page_0001.png`, `page_0002.png`, ...

### 3단계 — 자동 캡처 설정

| 항목 | 설명 | 기본값 |
|------|------|--------|
| 간격 (최소 ~ 최대) | 캡처 사이의 랜덤 대기 시간(초) | 1.0 ~ 3.0초 |
| 휴식 주기 | N 페이지마다 자동으로 쉼 | 30페이지 |
| 휴식 시간 | 쉬는 시간(초) | 10초 |
| 캡처 전 대기 | 페이지 넘긴 후 캡처하기까지 기다리는 시간 | 1.0초 |
| **총 페이지** | 이 수만큼 캡처하면 자동 종료 (0 = 무한) | 0 |

### 4단계 — 페이지 자동 넘김 설정 (DRM 앱 전용)

**클릭 위치 지정 (3초 후)** 버튼을 누르고, 3초 안에 마우스를  
ebook 뷰어의 **다음 페이지 버튼** 위로 이동하세요.  
이후 자동 모드에서 해당 위치를 자동으로 클릭해 페이지를 넘깁니다.

> 클릭 위치를 지정하지 않으면 페이지 넘김 없이 캡처만 반복합니다.

### 5단계 — 캡처 시작

#### 수동 모드

**수동 시작** 버튼을 클릭한 뒤, ebook 페이지를 넘길 때마다 단축키를 누릅니다.

| 단축키 | 동작 |
|--------|------|
| `Space` 또는 `→` | 현재 화면 캡처 & 저장 |
| `Q` 또는 `Esc` | 종료 |

#### 자동 모드

**자동 시작** 버튼을 클릭하면 설정된 간격으로 자동 클릭 + 캡처를 반복합니다.  
총 페이지 수를 입력하면 그 수만큼 찍고 자동 종료됩니다.

| 단축키 | 동작 |
|--------|------|
| `P` | 자동 캡처 일시정지 / 재개 |
| `Q` 또는 `Esc` | 종료 |

## PDF 변환

캡처한 이미지들을 하나의 PDF 파일로 합칩니다.  
상단 **PDF 변환** 탭으로 이동하세요.

| 항목 | 설명 |
|------|------|
| 이미지 폴더 | 변환할 이미지가 들어있는 폴더 |
| 저장할 PDF 파일 | 출력 PDF 경로 및 파일명 |
| 포함 확장자 | 변환할 파일 형식 (예: `png, jpg`) |
| 정렬 | **파일명** 순 (기본) 또는 **수정 시간** 순 |

**PDF 변환 시작** 버튼을 누르면 진행 바와 함께 변환되고,  
완료 시 저장 경로를 알려주는 팝업이 뜹니다.

## 팁

- 관리자 권한으로 실행해야 DRM 앱의 클릭 자동화가 동작합니다.
- ebook 뷰어를 전체화면으로 열고 영역을 지정하면 더 정확합니다.
- 자동 모드의 간격은 ebook 뷰어의 페이지 전환 속도에 맞게 조정하세요.
- 중간에 멈췄다 다시 시작할 경우 **시작 번호**를 이어지는 번호로 설정하세요.
- 저장 중 **폴더 열기** 버튼으로 저장 폴더를 바로 확인할 수 있습니다.

## EXE 빌드 (배포용)

Python 없이 실행 가능한 단일 `.exe` 파일로 빌드할 수 있습니다.

### 사전 준비

1. **Python 3.8 이상 설치**  
   [python.org](https://python.org) 에서 다운로드 후 설치  
   설치 시 **"Add Python to PATH"** 체크 필수

2. **라이브러리 설치**
   ```bash
   pip install -r requirements.txt
   ```

3. **아이콘 파일 준비** (선택)  
   `icon.ico` 파일을 프로젝트 폴더에 넣으면 exe 아이콘으로 사용됩니다.  
   PNG → ICO 변환:
   ```python
   from PIL import Image
   img = Image.open("icon.png").convert("RGBA").resize((256, 256))
   img.save("icon.ico", format="ICO", sizes=[(256,256),(128,128),(64,64),(32,32),(16,16)])
   ```

### 빌드

```
build.bat 더블클릭
```

완료되면 `dist\EbookCapture.exe` 생성됩니다.  
이 파일 하나만 있으면 Python 없이 어디서든 실행 가능합니다.

> `build/`, `dist/` 폴더는 빌드 산출물이며 Git에서 제외됩니다.
